# 复古机 — 第三方资源署名

## 内置试玩 Demo(均为开源/免费可再分发的 homebrew,按授权署名)
- **《Tobu Tobu Girl》**(GB)— Simon Larsen / Tangram Games。代码 MIT、美术/音乐 CC-BY 4.0。
  https://github.com/SimonLarsen/tobutobugirl · ROM:https://archive.org/details/TobuTobuGirlGame
- **《Libbet and the Magic Floor》**(GB)— Damian Yerrick (pinobatch)。Zlib 许可。
  https://github.com/pinobatch/libbet (release v0.08 `libbet.gb`)
- **《µCity》**(GBC)— Antonio Niño Díaz (AntonioND)。GPLv3。
  https://github.com/AntonioND/ucity (release v1.3 `ucity.gbc`)
- **《Nova the Squirrel》**(NES)— NovaSquirrel。GPL-3.0。
  https://github.com/NovaSquirrel/NovaTheSquirrel (release v1.0.6a `nova.nes`)

## 模拟器内核
**EmulatorJS**(开源)及其打包的 libretro 内核(fceumm / snes9x / gambatte / mgba / genesis_plus_gx / mupen64plus_next),自托管于 `emulatorjs/data/`。
- 项目:https://emulatorjs.org / https://github.com/EmulatorJS/EmulatorJS

## 用户 ROM
其余游戏 ROM 由用户自行本地上传,本应用不内置、不分发受版权保护的商业 ROM。
