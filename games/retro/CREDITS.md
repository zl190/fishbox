# 复古机 — 第三方资源署名

## 内置试玩 Demo
**《Tobu Tobu Girl》** — Game Boy homebrew,作者 Simon Larsen / Tangram Games。
- 官网:https://tangramgames.dk/tobutobugirl/
- 源码:https://github.com/SimonLarsen/tobutobugirl
- ROM 来源:https://archive.org/details/TobuTobuGirlGame (`tobu.gb`)
- 授权:代码 MIT,美术/音乐 CC-BY 4.0 —— 允许再分发,要求署名。本应用据此内置并署名。

## 模拟器内核
**EmulatorJS**(开源)及其打包的 libretro 内核(fceumm / snes9x / gambatte / mgba / genesis_plus_gx / mupen64plus_next),自托管于 `emulatorjs/data/`。
- 项目:https://emulatorjs.org / https://github.com/EmulatorJS/EmulatorJS

## 用户 ROM
其余游戏 ROM 由用户自行本地上传,本应用不内置、不分发受版权保护的商业 ROM。
